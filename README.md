                    重要　　プラグインを書いた行の後には　ソースコマンドを記述しないと反映されない
                    
    zsh　の　ログインシェルは　使えるまで時間がかかる
   debian　linux　では　wslのターミナルアプリを使用において
   
   開始してから　   1分くらいの待ち時間が　steeef　テーマのログインシェルになるまで時間がかかる。
   
   
   他にも　~/　から　ユーザディレクトリに戻る時にも　ログインシェルが反映されない。
